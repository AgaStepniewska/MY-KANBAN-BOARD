Kanban Board

This is very useful tool which will help you to organize your work. Basically it might contains 3 (or more) columns:

  • To do
  • Doing
  • Done

Thanks to this tool you can sort out all of your tasks. Kanban board is editable so you can customize it to fit your needs.
